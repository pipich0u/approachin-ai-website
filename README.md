# Approaching AI

一个现代化的 AI 产品展示网站，基于 React + TypeScript + Vite 构建，展示 KTransformers、Mooncake 等 AI 开源项目。

## 项目简介

Approaching AI 是一个响应式 Web 应用，提供完整的 PC 端和移动端体验，用于展示和推广先进的 AI 基础设施项目。

### 主要功能

- **首页展示** - 产品介绍、案例展示、技术咨询
- **KTransformers** - 高性能推理引擎展示页
- **Mooncake** - KV Cache 解决方案展示页
- **模型仓库** - AI 模型展示与管理
- **商务合作** - 合作流程与联系方式

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发运行

```bash
pnpm dev 
```

```bash
npm run dev
```

### 构建部署

```bash
# 生产构建
pnpm build

# 预览构建结果
pnpm preview
```
